# Integrador — Operaciones matriciales y mapeo a arquitectura

Programa de consola en Python que implementa operaciones matriciales básicas (2x2 y 3x3) y muestra su correspondencia al nivel de hardware (direcciones en memoria, traza de accesos y explicación del ciclo fetch–decode–execute para C[i,j]).

Uso rápido:

- Ejecuta: python3 main.py
- Sigue el menú para cargar matrices A y B (orden 2 o 3), y probar operaciones: suma, transpuesta, multiplicación, determinante.
Pruebas:

- Ejecuta las pruebas unitarias: python3 -m unittest discover -v
